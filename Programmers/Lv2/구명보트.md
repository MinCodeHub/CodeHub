문제를 제대로 안읽어서
구명 보트에 최대 2명씩 밖에 탈 수 없다는 사실을 
구글링 이후에 알아버림 ^^..

정답 코드
```java
import java.util.*;

class Solution {
    public int solution(int[] people, int limit) {
        int answer = 0;
        Arrays.sort(people);
        
        int idx = 0;
        
        for(int i = people.length -1; i>=idx; i--){
            if(people[i] + people[idx] <= limit){
                answer++;
                idx++;
            }else{
                answer++;
            }
        }
        
        
        return answer;
    }
}
```
