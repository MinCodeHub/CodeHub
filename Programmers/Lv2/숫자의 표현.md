효율성에서 모두 실패(시간초과)
```java

class Solution {
    public int solution(int n) {
        int answer = 0;
    
        
        for(int i = 1; i<=n; i++){
            int tmp = 0;
             for(int j = i; j<=n ; j++){
                 tmp += j;
                if(tmp == n){
                    answer++;
                    tmp = 0;
                    break;
            }
        }
        }
       
        return answer;
    }
}
```

왜 시간 초과?

1.  더한 수가 이미 n을 초과하면 break를 걸어야하나?  >> 정답!

최종 코드

```java
class Solution {
    public int solution(int n) {
        int answer = 0;
    
        
        for(int i = 1; i<=n; i++){
            int tmp = 0;
             for(int j = i; j<=n ; j++){
                 tmp += j;
                 
                if(tmp > n){
                    tmp = 0;
                    break;
                }else if(tmp == n){
                    answer++;
                    tmp = 0;
                    break;
            }
        }
        }
       
        return answer;
    }
}
```
